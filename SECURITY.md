# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest release | ✅ |
| Older releases | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability in Awebo, please report it responsibly.

**Do not open a public issue.**

Instead, email **hello@awebo.sh** with:

- A description of the vulnerability
- Steps to reproduce
- Affected version(s)
- Any potential impact

We will acknowledge your report within **48 hours** and aim to provide a fix or mitigation within **7 days** for critical issues.

## Scope

This policy covers the Awebo application and its build/release pipeline. Third-party dependencies are managed via Cargo and are subject to their own security policies.
